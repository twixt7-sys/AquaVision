Building and testing the 3D twin view — geometry rendering, the vertical slice interaction, and critically the PROVENANCE RENDERING that makes measured, interpolated, operator-supplied, and stale data look different.

The Non Negotiable

When this file feeds the twin, the whole view carries the illustrative_synthetic badge. And the interpolated layer must demonstrate the uncertainty-rendering requirement, because that requirement is the twin's core honesty feature and this synthetic data exists partly to let you build it.
